# Site Name

January 28, 2025

---
## Enable and Disable

- Default: true

```
theme:
  components:
    site_name: true
```

<button component-id="component-site-name" status="false"><code>true</code></button>
<button component-id="component-site-name" status="true"><code>false</code></button>

## Changing the fonts

```
theme:
  site_name_style: normal
```
